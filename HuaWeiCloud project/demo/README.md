# README

This demo is used to get you familiar with deep learning frameworks.

Unzip the compressed file and you will get such files:

```
demo/
| --- tf_demo/
|	  | --- keras_demo.py
|	  | --- run_tf_demo.sh
| --- torch_demo/
| 	  | --- torch_demo.py
|     | --- run_torch_demo.sh
| --- imgs/
|     | --- sample*.png
| --- mnist.npz
| --- torch_demo_notebook.ipynb
```

- `tf_demo/` contains an implementation with Tensorflow 2.x. Using `tf.keras` standard.
- `torch_demo/` contains an implementation with PyTorch 1.6.0.
- `imgs/` contains several sample images of MNIST dataset.
- `torch_demo_notebook.ipynb` contains jupyter interactive codes and tutorials, using PyTorch.
- `mnist.npz` is the MNIST handwritten digit dataset.
